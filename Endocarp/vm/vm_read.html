vm_read()

MIT Documentation for vm_read()

Example

/*
read data from task's address space
*/

vm_address_t address = 0x10086c000;
vm_address_t read_buffer = 0x0;
vm_size_t size = 4;

kern_return_t kret;

kret = vm_read_overwrite(mach_task_self(), //task for reading in
               address, //address where read begins
               size, //in size of read
               (vm_offset_t) &read_buffer, //medium for storing read data (out)
               &size); // out size

if (kret != KERN_SUCCESS) {
     return kret;
}
return KERN_SUCCESS;

Notes

The difference between vm_read and vm_read_overwrite() is how read data is handled.
