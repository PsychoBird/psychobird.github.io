Project Endocarp

An expansion of the MIT Mach Documentation


by PsychoBird - Last updated December 1, 2020

Project Endocarp's goal is to update the MIT Mach IPC Documentation with the updated mach_*_64 functions and to provide usage examples for most functions. A detailed and updated documentation would have aided in the development of my projects, so I'm launching Project Endocarp for others to more easily learn and code. This page is constantly being changed and expanded.
